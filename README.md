# Lidya Cultural Food Zone 🍽️

A beautifully designed, culturally rich restaurant website built for **Lidya Cultural Food Zone**. The website features a dynamic, responsive design that highlights the restaurant's authentic cultural experiences, menu offerings, gallery, and more.

## 🚀 Features

- **Dynamic Hero Section:** A parallax scrolling background paired with floating cultural elements (coffee beans, injera, stars, etc.).
- **Smooth Scroll Reveal Animations:** Content gracefully fades in as the user scrolls, creating a premium feel.
- **Interactive Menu:** A filterable menu section with categorized dishes and highlight tags.
- **Gallery Lightbox:** A masonry photo grid that allows users to view full-resolution images in a beautiful lightbox overlay.
- **Testimonials Carousel:** Auto-advancing quote carousel showcasing guest experiences.
- **Responsive Layout:** Fully functional and visually appealing on all devices, from mobile phones to large desktop screens.

## 🛠️ Tech Stack

- **React 18** - UI component framework
- **TypeScript** - Type-safe JavaScript
- **Vite 6** - Lightning-fast build tool and development server
- **Tailwind CSS 4** - Utility-first CSS styling for rapid UI development
- **Radix UI** - Unstyled, accessible UI primitives

## 📂 Project Structure

- `src/app/App.tsx`: The heart of the application containing all sections, data, and components.
- `src/imports/`: Contains all uploaded image assets like logos and restaurant photos.
- `src/styles/`: Contains all styling definitions including Tailwind configuration, fonts, and theme variables.
- `DOCUMENTATION.md`: In-depth documentation on modifying website content, replacing images, and customizing the design.

## 🏁 Getting Started

To get a local copy up and running, follow these simple steps.

### Prerequisites

Make sure you have [Node.js](https://nodejs.org/) installed on your machine.

### Installation

1. Clone the repository or extract the project files.
2. Open your terminal and navigate to the project directory:
   ```bash
   cd lidyaculturalfood
   ```
3. Install dependencies:
   ```bash
   npm install
   # or using pnpm
   pnpm install
   ```

### Running the App Locally

To start the development server, run:
```bash
npm run dev
# or using pnpm
pnpm dev
```
Open [http://localhost:5173](http://localhost:5173) to view it in the browser.

### Building for Production

To build the app for production to the `dist` folder, run:
```bash
npm run build
# or using pnpm
pnpm build
```
